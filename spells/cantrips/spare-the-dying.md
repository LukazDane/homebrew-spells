# Spare the Dying

- [x] Homebrew

***0th-level necromancy***

**Casting Time:** action

**Duration:** instantaneous

**Range:** touch

**Tags:** verbal, somatic

---

## Description
You touch a living creature that has 0 hit points.
The creature regains `1d4` hit points.

**Classes:** cleric, paladin, artificer

**Sources:** PHB, page 277
